# Admin Module — Complete Feature List
Last updated: 2026-05-30

## Accounts
- Financial Dashboard — Displays income, expense charts, and balance summaries
- Account Directory — Manage ledgers for cash, bank, and digital wallets
- Transaction Registry — Log payments, expenses, upload receipts, and void records
- Categories Customizer — Define transaction categories with custom colors and icons
- Fund Transfers — Transfer funds between internal accounts with auto balancing
- Reports Generator — Generate custom date-range income vs expense breakdowns
- Inter-account Transfers Registry `[ undocumented ]` — List and search internal cash flow transfers

## Academics
- Course Catalog — Displays active courses with details and basic metadata editor
- Curriculum Builder — Three-panel layout to organize weeks, days, and modules
- Learning Module Manager — Structure day curriculum with lessons, assignments, and quizzes
- Lesson Rich Editor — Create instructional pages with formatted rich text and media
- Quiz Builder — Design multiple-choice questions with answers and scoring settings
- Assignment Editor — Formulate descriptions, required file inputs, and point settings
- Assignment Submissions Inbox — Centralized queue to filter and search student work
- Assessment Detail Drawer — Expand text answers and download student submission attachments
- Assessment Feedback Console — Approve student assignments or request revisions with notes
- Standalone Tasks Catalog — Manage independent assignments not bound to any course
- Custom Task Scope — Assign standalone assignments directly to students
- Standalone Grading Queue — Central inbox to grade and review independent assignments

## Employee
- Employee Directory — Grid listing of staff with department and status filters
- Detailed Employee Profile — Displays personal, contact, employment details, and documents
- Document Vault — Upload and secure Aadhaar, PAN, and certificate files
- Employee Creation Form — Register staff, crop avatars, and check duplicate credentials
- Temporary Password Vault — Display login keys for manual employee welcome delivery
- Department Management `[ undocumented ]` — Define organizational departments and assign department heads

## Attendance
- Daily Attendance Summaries — Summary cards displaying real-time counts of employee statuses
- Excel-style Timesheet Grid — View and scroll calendar sheets mapping employee daily attendance
- Cell Marking Modal — Log daily attendance status, audit notes, and history
- Attendance Audit Trail — Track marked timestamps and editor credentials for audit checks
- Payroll Locking Safety — Lock marked attendance days once monthly payroll disbursed

## Payroll
- Monthly Payroll Dashboard — View obligation summaries and employee salary sheets monthly
- Automatic Calculation Engine — Computes Net Payable salary adjusting for LOP and absences
- Sandwich Leave Logic — Enforce LOP calculations for leave blocks spanning weekends
- Payroll Lifecycle Pipeline — Transition payroll runs through Draft, Approved, and Disbursed
- Real-time Attendance Synchronization — Recalculates LOP deductions automatically when attendance changes
- Financial Ledger Disbursal — Debits payroll from accounts, logs expenses, and locks logs
- Payslips Vault — View employee attendance summaries and LOP breakdown panels
- Payslip PDF Downloader — Print-optimized template to download payslips as PDF files

## Students
- Student Directory — Register and display student profiles with enrollment details
- Student Profile Dashboard — View enrolled courses, progress bars, and syllabus accordion
- Course Submissions Tab — Review assignment work and check graded status logs
- Student Action Sidebar — Assign courses or standalone tasks directly to students
- Standalone Submissions History — Track and redirect student standalone assignment evaluations

## Fee Management `[ undocumented ]`
- Overdue Payments Dashboard `[ undocumented ]` — Track overdue student installments and filter by course
- Record Payments Console `[ undocumented ]` — Collect installment dues and adjust remaining balances
- Payment History Registry `[ undocumented ]` — View logged payments, waivers, and adjustments for students
- Fee Receipt Generator `[ undocumented ]` — Preview payments history and download native PDF receipts
- Fee Scheme Configurator `[ undocumented ]` — Set one-time versus installment course pricing parameters
- Account Waivers and Adjustments `[ undocumented ]` — Adjust outstanding dues and apply fee waivers directly

## Security & Access Control
- Admins Directory — Displays administrative accounts, security roles, and employee connections
- Credentials Configurator — Set up admin passwords, roles, and status flags
- Roles Registry — Define custom roles and manage existing security role labels
- Permissions Matrix — Granular toggle matrix gating page access and CRUD permissions
