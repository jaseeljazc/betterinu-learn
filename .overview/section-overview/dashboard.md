# Admin Dashboard Section Overview

The Admin Dashboard provides a high-level summary of school-wide activity and metrics. It acts as the home screen for administrators to quickly review numbers and jump to recent activities.

---

## Key Features

### 1. Metric Summary Cards
*   **Total Students:** Shows the absolute count of registered students in the LMS.
*   **Active Courses:** Displays the number of published and active courses.
*   **Total Assignments:** Tracks the overall volume of course assignments assigned to students.
*   Each card features a colored icon representing the metric to enhance visual hierarchy.

### 2. Latest Submissions Feed
*   **Unified Activity Table:** Merges recent student submissions from both **course assignments** and **standalone tasks** in real time.
*   **Details Row:** Each submission shows:
    *   Student Name (linking to their details).
    *   Assignment/Task Title.
    *   Type (Course Assignment or Standalone).
    *   Status badge:
        *   `Pending` (amber) - Under review.
        *   `Approved` (green) - Accepted.
        *   `Revise` (red) - Sent back for corrections.
    *   Relative submission timestamp.
*   **Quick Links:** Provides deep links to the full grading queues (e.g., Submissions section).
