# Administrator Management Section Overview

The Administrator Management section controls access to the admin panel by managing administrator logins, security roles, and link mappings to employee records.

---

## Key Features

### 1. Admin Directory
*   **Admins Table:** Lists all admin panel users with their full name, email, linked employee profile, status badge, and assigned security role.
*   **Role Badges:** Quick identification of permissions levels (e.g., Super Admin, Account Manager, Academic Coordinator).

### 2. Admin Form (Create & Edit)
*   **Credentials Setup:** Allows setting up admin credentials (Email, Full Name, and Password).
*   **Role Assignment:** Dropdown list to assign a predefined security Role to the administrator.
*   **Employee Linking:** Map the administrator login account directly to an **Employee Record** in the HR system. This link enables features like self-service profile updates and payslip views.
*   **Account Controls:** Mark administrators as active/inactive or pending to restrict system access.
