# Standalone Tasks Section Overview

The Standalone Tasks section handles assignments that are not bound to a specific course curriculum (independent assignments). Admins use this to manage standalone tasks, assign them directly to students, and grade submissions.

---

## Key Features

### 1. Standalone Tasks Catalog
*   **Active Directory:** Lists all standalone tasks with details on difficulty level, title, description, scope (e.g. `common` or `course-specific`), and points.
*   **Task Management:** Create, edit, and delete standalone tasks.

### 2. Student Assignment
*   **Assignment Controls:** Select a task and assign it to multiple students.
*   **Task Scope:**
    *   **Common Tasks:** Can be assigned to any student in the system.
    *   **Course Tasks:** Can only be assigned to students enrolled in the corresponding course.

### 3. Submission Statistics
*   **Tally Indicators:** Tracks submission numbers, showing the count of:
    *   `Submitted` tasks.
    *   `Approved` tasks.
    *   `Pending` reviews.
    *   `Revisions Requested`.

### 4. Grading Interface
*   **Unified Review Queue:** Lists all standalone task submissions.
*   **Feedback & Grading Dialog:** Renders text responses and attachments, allowing admins to approve/reject the submission and enter custom evaluation notes.
